## Pet Mall Website

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/3fdfd031f2144e67ad6760ba16cf3d0e/ghb20250918) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can be used to run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): regular users, administrators.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. Encyclopedia Information
5. Forum
6. Message Board
7. Products
8. Personal Center
9. Delivery Address
10. Product Collection
11. Product Orders
12. Background Management
13. Shopping Cart


**Administrator Functions:**
1. Login
2. Personal Center (change password, personal information)
3. Encyclopedia Information Management
4. Basic Data Management (encyclopedia type management, member level type management, product type management)
5. Forum Management
6. Message Board Management
7. Product Management, Product Review Management, Product Collection Management, Product Order Management
8. User Management
9. Carousel Image Information
